# Name of project
A productivity based website that will auto-generate a calendar based on
user productivity, personalization settings, and task details.

## Features
 - Creates tasks with:
   - Due dates
   - Importance (1-5, with 5 being the most important)
   - Estimated time needed
   - Difficulty (Easy/Medium/Hard)
   - Tags (Homework, Study, Work, etc.)
 - Set productivity levels:
   - Most productive in morning, afternoon, evening, or late night
   - Custom working time blocks
 - Creates a 7-day working schedule
   - Prioritizes tasks based on task details using equations
   - Edit, delete, and mark tasks as complete
   - Includes breaks to maintain productivity
